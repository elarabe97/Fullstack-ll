# Manual de Usuario - Level-Up Gamer

## 1. Instalación y Ejecución
### Backend
1. Abrir carpeta `backend`.
2. Ejecutar `mvn spring-boot:run` (o usar IDE como IntelliJ/Eclipse).
3. El servidor iniciará en `http://localhost:8080`.
4. **Usuarios de Prueba**:
   - Admin: `admin@levelupgamer.com` / `admin123`
   - Cliente Duoc: `cliente@duoc.cl` / `duoc123`

### Frontend
1. Abrir carpeta `frontend`.
2. Ejecutar `npm install` para instalar dependencias.
3. Ejecutar `npm run dev`.
4. Abrir navegador en `http://localhost:5173`.

## 2. Funcionalidades
### Registro e Inicio de Sesión
- Acceda a "Registro" para crear cuenta.
- **Nota**: Si usa un correo `@duoc.cl`, obtendrá 20% de descuento en todas sus compras.
- Acceda a "Login" para ingresar.

### Catálogo
- Visualice los juegos disponibles.
- Use el botón "Agregar" para sumar al carrito.

### Carrito de Compras
- Revise sus productos.
- Verá el desglose de Subtotal, Descuento (si aplica) y Total.
- Al finalizar compra, ganará 1 punto por cada $1.000 pesos.

### Comunidad
- Lea reseñas de otros usuarios.
- Si está logueado, puede dejar su propia reseña con calificación de 1 a 5 estrellas.
